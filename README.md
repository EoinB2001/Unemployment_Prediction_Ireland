Unemployment Prediction in Ireland

This repository contains the code used in my dissertation:

"Predicting Unemployment in Ireland Using Machine Learning Techniques and Macroeconomic Indicators"

Overview

This project applies machine learning models to predict unemployment rates in Ireland using quarterly macroeconomic data.

Models Used:
Linear Regression
Random Forest
Gradient Boosting

Data Sources:
Central Statistics Office (CSO)
World Bank
FRED (Federal Reserve)

Features:
Lagged variables (up to 4 quarters)
Growth rates for GDP, CPI, interest rate, exports

Evaluation Metric:
Root Mean Squared Error (RMSE)

This repository supports the analysis presented in my dissertation.
